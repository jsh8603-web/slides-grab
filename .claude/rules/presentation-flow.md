## 단계 요약

- **Step 0**: 소스 확인 (번호 선택형 질문 필수)
- **Step 1**: 폴더 생성 + 아웃라인 작성 → 사용자 승인
- **Step 1.5A**: Marp 초안 확인 (건너뛰기 금지)
- **Step 1.5B**: NanoBanana 이미지 자동 생성
- **Step 2**: HTML 슬라이드 생성 + Preflight 검증
- **Step 2.5**: 자동 검증 + COM 300DPI 시각 확인
- **Step 3**: 비주얼 에디터 실행
- **Step 4**: 수정 반복
- **Step 5**: 출력 형식 선택 (번호 선택형 질문 필수)
- **Step 6**: PPTX 생성 + 검사
- **Step 7**: PDF 생성
